# Dog&Cat Petstore

Dog and Cat Petstore is an application that Lists pets available at our store and are available for adoption.


## API Endpoints

#### Base Url

```http
  https://mocki.io/v1
```

#### Get Cats List

```http
  GET /77290441-d8e9-483f-80a2-a336a3364bbe

Method: GET

```
#### Get Dogs List

```http
  GET /442a4801-5715-474a-b6fd-7548ab2f8d20

Method: GET

```
